\section{Query agent và bộ dữ liệu}
\subsection{Bộ dữ liệu}
Sử dụng bộ dữ liệu giả sinh bằng thư viện Faker của Python để tạo ra một tập dữ liệu gồm 10 test user, mỗi user 
có 10 categories và 100 giao dịch liên quan. Xem thêm tại \textbf{~\ref{sec:database}}.

\subsection{Query agent}
Agent có các tool như: truy vấn, chấm điểm kết quả trả về, tối ưu câu lệnh,... Các tool này sẽ được agent sử dụng để sinh câu lệnh tối ưu
và trả kết quả query về cho supervisor agent.
